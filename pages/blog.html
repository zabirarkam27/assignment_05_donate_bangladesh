What is the Document Objet Model (DOM)?

The Document Object Model (DOM) is a programming interface for web documents. It represents the structure of an HTML or XML document as a tree of objects, where each element, attribute, and piece of content is a node. The DOM allows developers to access, modify, and manipulate a webpageâ€™s structure, style, and content dynamically using JavaScript, enabling interactive and dynamic web experiences.

How do you select an element from the DOM?

To select an element from the DOM, you can use JavaScript methods like document.getElementById() to select an element by its ID, document.querySelector() to select the first element matching a CSS selector, or document.querySelectorAll() to select all elements matching a CSS selector. Other methods include document.getElementsByClassName() for selecting elements by class and document.getElementsByTagName() for selecting elements by tag name. These methods allow you to manipulate or interact with the selected elements programmatically.

What is event delegation in the context of the DOM, and why is it useful?

Event delegation is a technique in the DOM where a single event listener is added to a parent element to manage events on its child elements, even those added dynamically. This works because events bubble up from the target element to its ancestors. It is useful as it improves performance by reducing the number of event listeners, simplifies code, and ensures event handling works for dynamically added elements. For example, instead of adding a click listener to every button in a list, you attach it to the list container and check the event target.

How do you manipulate an element's attributes and styles using the DOM?

To manipulate an element's attributes and styles using the DOM, you can use JavaScript. For attributes, use methods like setAttribute(), getAttribute(), and removeAttribute(), or directly modify properties such as element.id or element.href. For styles, access the style property of an element and assign values, e.g., element.style.color = "red" or element.style.fontSize = "16px". You can also manipulate classes using methods like classList.add(), classList.remove(), or classList.toggle() to dynamically change the element's appearance.
